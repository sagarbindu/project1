Skill 3-
arduino uno

Arduino Uno is a microcontroller board developed by Arduino.cc and based on Atmega328. It is an open-source platform, means the boards and software are readily available and anyone can modify and optimize the boards for better functionality. The software used for Arduino devices is called IDE (Integrated Development Environment) which is free to use and required some basic skills to learn it. It can be programmed using C and C++ language.
